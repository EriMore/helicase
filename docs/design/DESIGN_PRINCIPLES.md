# Design Principles

1. **Open on wonder.** The default state is a cinematic protein sky, not a control panel.
2. **Let science carry the color.** Confidence, family, and generation have distinct meanings.
3. **Keep chrome quiet.** Controls should frame the object rather than compete with it.
4. **Use depth to explain scale.** Map, structure, domain, residue, and atom should feel like one continuous zoom.
5. **Make uncertainty beautiful and unmistakable.** Fog is a semantic state, not decoration.
6. **Treat typography as instrumentation.** IDs, scores, and sequences use precise readable conventions.
7. **Design for the pause.** The strongest transitions need time to resolve and be understood.
8. **Borrow patterns, not surfaces.** External references inform motion and composition; Helicase must remain original.
9. **Respect accessibility.** Meaning must not depend on color alone; motion needs reduction and interruption paths.
10. **Every visual flourish earns its place.** If it does not communicate state, scale, evidence, or wonder, remove it.
