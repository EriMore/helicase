# Visual Language

## Composition

Use a large uninterrupted field with a small number of anchored zones: the scene, a quiet identity rail, a contextual copilot, and transient evidence. Panels should feel like instruments projected into space, not cards stacked over a dashboard.

## Hierarchy

1. Molecular object or map region.
2. Current action and state.
3. Scientific qualifier: source, confidence, or prediction status.
4. Optional explanation and controls.

Never let navigation outrank the object being explored.

## Materials and depth

The base material is optical darkness: soft black, barely visible grain, and depth created through falloff rather than borders. Structures are luminous but not glossy. Generation uses a restrained violet signal; confidence uses a cold blue-to-amber scale; interface chrome is nearly neutral.

## Information density

The cold open is sparse. Density increases only after selection. Detail should arrive at the scale where it becomes useful: universe → family → protein → domain → residue.

## What Helicase rejects

- Generic SaaS cards and pill-heavy control bars.
- Neon cyberpunk as a substitute for scientific meaning.
- Decorative particle fields that do not encode scale or state.
- Fake laboratory dashboards and unnecessary metric walls.
- Literal galaxy imagery that competes with the molecular subject.

## Moodboard in words

Observatory at midnight; polished optical instrument; electron haze; museum-grade specimen case; quiet aerospace console; a black room where evidence slowly lights itself.
