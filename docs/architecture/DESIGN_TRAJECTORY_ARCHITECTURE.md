# Design trajectory architecture

Helicase reveals attributable, precomputed design evidence through the existing SceneController boundary. The copilot's `start_design` tool is an eligibility mapper, not a sequence generator: it resolves a request to `proteinmpnn-6ehb-example-6` only when the selected target is UniProt A5F934 / experimental PDB 6EHB.

The public run is coherent from target structure to ProteinMPNN candidates. It does not contain RFdiffusion frames, predicted candidate structures, interface metrics, confidence values, or wet-lab validation. Those stages remain explicit evidence-gate states (`DesignPanel.tsx`'s `backbone`/`fold` beats) and are never visually implied as completed — see `docs/handoff/DESIGN_DELTA.md` §11.

Playback is a **continuous, on-screen, in-flight spatial sequence** — `design.progress` is a single `0..1` value across 6 labeled beats (Target & Site, Backbone Candidates, Sequence Design Candidates, Predicted Fold & Metrics, Compare, Resolve), driven by `SEEK_DESIGN`/`SET_DESIGN_PLAYBACK` in `src/domain/atlas.ts`, not a discrete stage index advanced by Next/Back clicks. The user can play/pause, scrub the continuous timeline directly, or jump to any beat's start; a real character-level sequence diff between the two ProteinMPNN candidates renders in the `sequence` and `compare` beats. This intentionally supersedes an earlier discrete 9-stage click-through design — see `docs/handoff/DESIGN_DELTA.md` §1 for the explicit rationale. Beat timing is a presentation affordance for inspecting imported artifacts, not a claim about compute time. The player never interpolates absent coordinates or labels a prediction as experimentally validated.

The fixture records source URLs, repository path, method, version/commit, input and output identities, license, timestamp and limitations at trajectory, stage and candidate level. The source is the official [ProteinMPNN example 6 output](https://github.com/dauparas/ProteinMPNN/tree/main/outputs/example_6_outputs) for PDB [6EHB](https://www.rcsb.org/structure/6EHB), imported at `public/data/design/proteinmpnn-6ehb.json`.
